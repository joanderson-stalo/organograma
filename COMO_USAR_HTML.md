# Como Baixar o Organograma em HTML Puro

## Arquivo Criado

Foi criado o arquivo: **organograma-standalone.txt**

## Como Usar

### Passo 1: Baixar o arquivo
Baixe o arquivo `organograma-standalone.txt` para seu computador.

### Passo 2: Renomear
Renomeie o arquivo de `.txt` para `.html`:
- **organograma-standalone.txt** → **organograma.html**

### Passo 3: Abrir no navegador
Dê um duplo clique no arquivo `organograma.html` ou arraste ele para seu navegador (Chrome, Firefox, Edge, Safari, etc.)

## Funcionalidades

O arquivo HTML inclui:
- ✅ Organograma completo e interativo
- ✅ Todas as pessoas e cargos
- ✅ Páginas de detalhes de cada área
- ✅ Informações do CTO
- ✅ Estilos CSS embutidos
- ✅ JavaScript para navegação
- ✅ Totalmente standalone (não precisa de internet)

## Vantagens

- **Um único arquivo**: Tudo em um arquivo HTML
- **Sem dependências**: Não precisa instalar nada
- **Offline**: Funciona sem internet
- **Fácil compartilhar**: Envie por email ou WhatsApp
- **Responsivo**: Funciona em desktop e mobile

## Compartilhar

Você pode:
1. Enviar o arquivo por email
2. Compartilhar via WhatsApp/Telegram
3. Colocar em um pen drive
4. Hospedar em qualquer servidor web
5. Adicionar ao Google Drive/Dropbox

## Hospedar Online (Grátis)

### Opção 1: GitHub Pages
1. Crie um repositório no GitHub
2. Faça upload do arquivo como `index.html`
3. Ative GitHub Pages nas configurações
4. Seu site estará online!

### Opção 2: Netlify Drop
1. Acesse https://app.netlify.com/drop
2. Arraste o arquivo HTML
3. Pronto! Seu site está online instantaneamente

### Opção 3: Surge.sh
```bash
npm install -g surge
surge organograma.html
```

## Personalizações

Para editar, abra o arquivo em qualquer editor de texto:
- Notepad (Windows)
- TextEdit (Mac)
- VS Code
